Depuis la récente brèche de sécurité au niveau des clés de chambres, nous avons accru nos mesures de sécurité à l'accueil. Cependant, nous sommes conscients qu'un risque demeure au niveau des clés de chambres volées.

Vous pouvez utiliser le formulaire ci-dessous pour déterminer si vous êtes à risque. Si c'est le cas, n'hésitez pas à venir à la réception pour obtenir une nouvelle clé de chambre.

Le code de chambre débute par CBG: suivi du nombre inscrit sur votre porte.
La clé de chambre est une séquence de 16 caractères.

Par souci de transparence, vous pouvez consulter le code source de notre site web afin de valider par vous même la qualité de nos nouvelles mesures de sécurité.
