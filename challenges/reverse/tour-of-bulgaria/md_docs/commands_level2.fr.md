# Commandes V0.1

## Commande Print Flag

ID de Commande : 0x99

### Description :
Afficher le flag.

### Arguments :
| Index d'Argument | Type | Nom | Description |
|---------|---------|----------|----------|
| **1** | Integer | Key | Clé pour décrypter le flag |
| **2** | Register | Register | Registre pour stocker la clé |
